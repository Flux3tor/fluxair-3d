# FluxAir 3D

---

**FluxAir 3D is a browser-based 3D air-drawing experience** powered by real-time hand tracking. Move your hand to control a virtual pen, pinch to draw, and create glowing strokes that stay floating in 3D space. No VR headset or special hardware required, just a modern browser and your webcam.
